# Daily Prayer Tracker (GUI) 🕌

A lightweight desktop application built with Python and Tkinter designed to help users track and log their daily 5-time prayers (Fajr, Dhuhr, Asr, Maghrib, Isha) along with congregation location details.

---

## ✨ Features

- **Graphical User Interface (GUI)**: Clean, simple Tkinter interface replacing traditional command-line interactions.
- **Congregation Tracking**: Separate options to log whether prayers were offered individually or at the Masjid.
- **Local Persistence**: Automatically appends daily records with timestamps to a formatted `namaz.txt` log file.
- **Input Validation**: Ensures user details are entered before saving.
- **Auto Reset**: Clears checkmarks automatically after successfully saving entries.

---

## 🛠️ Prerequisites

- **Python 3.x**
- **Tkinter** *(pre-installed with standard Python distributions on Windows and macOS)*
